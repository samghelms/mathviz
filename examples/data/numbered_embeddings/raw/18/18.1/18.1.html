§18.1(i) Special Notation

Referenced by:

(For other notation see Notation for the Special Functions.)

x,y real variables.
z(=x+i⁢y) complex variable.
q real variable such that 0<q<1, unless stated otherwise.
ℓ,m nonnegative integers.
n nonnegative integer, except in §18.30.
N positive integer.
δ⁡(x-a) Dirac delta (§1.17).
δ arbitrary small positive constant.
pn⁡(x) polynomial in x of degree n.
p-1⁡(x) 0.
w⁡(x) weight function (≥0) on an open interval (a,b).
wx weights (>0) at points x∈X of a finite or countably infinite subset of ℝ.
OP’s orthogonal polynomials.

x-Differences

Defines:
∇x: backward difference and δx: central difference
Keywords:
central differences in imaginary direction, difference operators
See also:
info for 18.1(i)

Forward differences:

Δx(f⁡(x)) =f⁡(x+1)-f⁡(x),
Δxn+1(f⁡(x)) =Δx(Δxn(f⁡(x))).

Backward differences:

∇x(f⁡(x)) =f⁡(x)-f⁡(x-1),
∇xn+1(f⁡(x)) =∇x(∇xn(f⁡(x))).

Central differences in imaginary direction:

δx(f⁡(x)) =(f⁡(x+12⁢i)-f⁡(x-12⁢i))/i,
δxn+1(f⁡(x)) =δx(δxn(f⁡(x))).

§18.1(ii) Main Functions

Permalink:
http://dlmf.nist.gov/18.1.ii
See also:
info for 18.1

The main functions treated in this chapter are:

Classical OP’s

Keywords:
Chebyshev polynomials, Hermite polynomials, Jacobi polynomials, Laguerre polynomials, Legendre polynomials, classical orthogonal polynomials, ultraspherical polynomials
See also:
info for 18.1(ii)

  • Jacobi: Pn(α,β)⁡(x).

  • Ultraspherical (or Gegenbauer): Cn(λ)⁡(x).

  • Chebyshev of first, second, third, and fourth kinds: Tn⁡(x), Un⁡(x), Vn⁡(x), Wn⁡(x).

  • Shifted Chebyshev of first and second kinds: Tn*⁡(x), Un*⁡(x).

  • Legendre: Pn⁡(x).

  • Shifted Legendre: Pn*⁡(x).

  • Laguerre: Ln(α)⁡(x) and Ln⁡(x)=Ln(0)⁡(x). (Ln(α)⁡(x) with α≠0 is also called Generalized Laguerre.)

  • Hermite: Hn⁡(x), Hen⁡(x).

Hahn Class OP’s

Keywords:
Hahn class orthogonal polynomials
See also:
info for 18.1(ii)

  • Hahn: Qn⁡(x;α,β,N).

  • Krawtchouk: Kn⁡(x;p,N).

  • Meixner: Mn⁡(x;β,c).

  • Charlier: Cn⁡(x;a).

  • Continuous Hahn: pn⁡(x;a,b,a¯,b¯).

  • Meixner–Pollaczek: Pn(λ)⁡(x;ϕ).

Wilson Class OP’s

Keywords:
Wilson class orthogonal polynomials
See also:
info for 18.1(ii)

  • Wilson: Wn⁡(x;a,b,c,d).

  • Racah: Rn⁡(x;α,β,γ,δ).

  • Continuous Dual Hahn: Sn⁡(x;a,b,c).

  • Dual Hahn: Rn⁡(x;γ,δ,N).

q-Hahn Class OP’s

See also:
info for 18.1(ii)

  • q-Hahn: Qn⁡(x;α,β,N;q).

  • Big q-Jacobi: Pn⁡(x;a,b,c;q).

  • Little q-Jacobi: pn⁡(x;a,b;q).

  • q-Laguerre: Ln(α)⁡(x;q).

  • Stieltjes–Wigert: Sn⁡(x;q).

  • Discrete q-Hermite I: hn⁡(x;q).

  • Discrete q-Hermite II: h~n⁡(x;q).

Askey–Wilson Class OP’s

See also:
info for 18.1(ii)

  • Askey–Wilson: pn⁡(x;a,b,c,d|q).

  • Al-Salam–Chihara: Qn⁡(x;a,b|q).

  • Continuous q-Ultraspherical: Cn⁡(x;β|q).

  • Continuous q-Hermite: Hn⁡(x|q).

  • Continuous q-1-Hermite: hn⁡(x|q)

  • q-Racah: Rn⁡(x;α,β,γ,δ|q).

Other OP’s

See also:
info for 18.1(ii)

  • Bessel: yn⁡(x;a).

  • Pollaczek: Pn(λ)⁡(x;a,b).

Classical OP’s in Two Variables

See also:
info for 18.1(ii)

  • Disk: Rm,n(α)⁡(z).

  • Triangle: Pm,nα,β,γ⁡(x,y).

In Szegő (1975, §4.7) the ultraspherical polynomials Cn(λ)⁡(x) are denoted by Pn(λ)⁡(x). The ultraspherical polynomials will not be considered for λ=0. They are defined in the literature by C0(0)⁡(x)=1 and

18.1.1 Cn(0)⁡(x)=2n⁢Tn⁡(x)=2⁢(n-1)!(12)n⁢Pn(-12,-12)⁡(x),
n=1,2,3,….
Symbols:
Tn⁡(x): Chebyshev polynomial of the first kind, Pn(α,β)⁡(x): Jacobi polynomial, (a)n: Pochhammer’s symbol (or shifted factorial), !: factorial (as in n!), Cn(λ)⁡(x): ultraspherical (or Gegenbauer) polynomial, n: nonnegative integer and x: real variable

In Mason and Handscomb (2003), the definitions of the Chebyshev polynomials of the third and fourth kinds Vn⁡(x) and Wn⁡(x) are the converse of the definitions in this chapter.
