§33.10(i) Large ρ

Notes:
See Yost et al. (1936).
Keywords:
Coulomb functions: variables ρ,η
Permalink:

As ρ→∞ with η fixed,

33.10.1 Fℓ⁡(η,ρ) =sin⁡(θℓ⁡(η,ρ))+o⁡(1),
Gℓ⁡(η,ρ) =cos⁡(θℓ⁡(η,ρ))+o⁡(1),
Symbols:
θℓ⁡(η,ρ): phase of Coulomb functions, cos⁡z: cosine function, Gℓ⁡(η,ρ): irregular Coulomb radial function, o⁡(x): order less than, Fℓ⁡(η,ρ): regular Coulomb radial function, sin⁡z: sine function, ℓ: nonnegative integer, ρ: nonnegative real variable and η: real parameter

§33.10(ii) Large Positive η

Notes:
See Fröberg (1955), Humblet (1984), Humblet (1985, Eqs. 2.10a,b). For (33.10.6) use (33.2.5), (33.2.10), and §5.11(i).

As η→∞ with ρ fixed,

33.10.3 Fℓ⁡(η,ρ) ∼(2⁢ℓ+1)!⁢Cℓ⁡(η)(2⁢η)ℓ+1⁢(2⁢η⁢ρ)1/2⁢I2⁢ℓ+1⁡((8⁢η⁢ρ)1/2),
Gℓ⁡(η,ρ) ∼2⁢(2⁢η)ℓ(2⁢ℓ+1)!⁢Cℓ⁡(η)⁢(2⁢η⁢ρ)1/2⁢K2⁢ℓ+1⁡((8⁢η⁢ρ)1/2).
Symbols:
Cℓ⁡(η): normalizing constant for Coulomb radial functions, ∼: asymptotic equality, !: factorial (as in n!), Gℓ⁡(η,ρ): irregular Coulomb radial function, Iν⁡(z): modified Bessel function of the first kind, Kν⁡(z): modified Bessel function of the second kind, Fℓ⁡(η,ρ): regular Coulomb radial function, ℓ: nonnegative integer, ρ: nonnegative real variable and η: real parameter

§33.10(iii) Large Negative η

As η→-∞ with ρ fixed,

33.10.7 Fℓ⁡(η,ρ) =(2⁢ℓ+1)!⁢Cℓ⁡(η)(-2⁢η)ℓ+1⁢((-2⁢η⁢ρ)1/2⁢J2⁢ℓ+1⁡((-8⁢η⁢ρ)1/2)+o⁡(|η|1/4)),
Gℓ⁡(η,ρ) =-π⁢(-2⁢η)ℓ(2⁢ℓ+1)!⁢Cℓ⁡(η)⁢((-2⁢η⁢ρ)1/2⁢Y2⁢ℓ+1⁡((-8⁢η⁢ρ)1/2)+o⁡(|η|1/4)).

Jν⁡(z): Bessel function of the first kind, Yν⁡(z): Bessel function of the second kind, Cℓ⁡(η): normalizing constant for Coulomb radial functions, !: factorial (as in n!), Gℓ⁡(η,ρ): irregular Coulomb radial function, o⁡(x): order less than, Fℓ⁡(η,ρ): regular Coulomb radial function, ℓ: nonnegative integer, ρ: nonnegative real variable and η: real parameter
