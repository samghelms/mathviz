§33.21(i) Limiting Forms

Notes:
See Seaton (2002a, Eqs. 104, 107), or apply (13.14.21) to (33.16.9).
Keywords:
Coulomb functions: variables r,ϵ
Permalink:
http://dlmf.nist.gov/33.21.i
See also:
info for 33.21

We indicate here how to obtain the limiting forms of f⁡(ϵ,ℓ;r), h⁡(ϵ,ℓ;r), s⁡(ϵ,ℓ;r), and c⁡(ϵ,ℓ;r) as r→±∞, with ϵ and ℓ fixed, in the following cases:

  1. (a)

    When r→±∞ with ϵ>0, Equations (33.16.4)–(33.16.7) are combined with (33.10.1).

  2. (b)

    When r→±∞ with ϵ<0, Equations (33.16.10)–(33.16.13) are combined with

    33.21.1 ζℓ⁡(ν,r) ∼e-r/ν⁢(2⁢r/ν)ν,
    ξℓ⁡(ν,r) ∼er/ν⁢(2⁢r/ν)-ν,
    r→∞,
    Symbols:
    ∼: asymptotic equality, e: base of exponential function, ℓ: nonnegative integer, r: real variable, ν: parameter, ζℓ⁡(ν,r): function and ξℓ⁡(ν,r): function

    33.21.2 ζℓ⁡(-ν,r) ∼er/ν⁢(-2⁢r/ν)-ν,
    ξℓ⁡(-ν,r) ∼e-r/ν⁢(-2⁢r/ν)ν,
    r→-∞.
    Symbols:
    ∼: asymptotic equality, e: base of exponential function, ℓ: nonnegative integer, r: real variable, ν: parameter, ζℓ⁡(ν,r): function and ξℓ⁡(ν,r): function

    Corresponding approximations for s⁡(ϵ,ℓ;r) and c⁡(ϵ,ℓ;r) as r→∞ can be obtained via (33.16.17), and as r→-∞ via (33.16.18).

  3. (c)

    When r→±∞ with ϵ=0, combine (33.20.1), (33.20.2) with §§10.7(ii), 10.30(ii).
